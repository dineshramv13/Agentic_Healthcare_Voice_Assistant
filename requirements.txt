# =====================================================================
# IMPORTANT — Phase 4 has SYSTEM-LEVEL (non-pip) dependencies:
#
# 1. ffmpeg (required by openai-whisper for audio decoding):
#      Ubuntu/Debian:  sudo apt update && sudo apt install ffmpeg
#      macOS:          brew install ffmpeg
#      Windows:        choco install ffmpeg   (or scoop install ffmpeg)
#
# 2. PortAudio (required by sounddevice for microphone access):
#      Ubuntu/Debian:  sudo apt install portaudio19-dev
#      macOS:          brew install portaudio
#      Windows:        no extra step needed (bundled in the sounddevice wheel)
#
# 3. Python version: Whisper does NOT support Python 3.13+ as of this
#    writing. Use Python 3.10, 3.11, or 3.12 for this project.
#
# 4. On Linux, pyttsx3 uses espeak as its backend — if TTS produces no
#    audio: sudo apt install espeak
#
# First run of voice/stt.py downloads the Whisper "base" model (~150MB)
# from Hugging Face automatically, cached at ~/.cache/whisper/. After
# that, everything (STT + TTS) runs fully offline, no API key, no cost.
# =====================================================================
